# 1000 Challenge — Job Hunt Tracker
### Offline-first PWA · Installable on iOS & Android · Works in any browser

---

## What this is

A progressive web app to track your 1000 job applications in 2026.  
- **Offline-first** — all data lives in your browser's IndexedDB. Works with no internet.
- **Installable** — add to home screen on iOS, Android, or desktop like a native app.
- **Exportable** — download all your applications as a `.csv` file for Excel.

---

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Generate app icons
npm install --save-dev sharp
node scripts/generate-icons.mjs

# 3. Run in development
npm run dev

# 4. Build for production
npm run build

# 5. Preview production build (test PWA / offline)
npm run preview
```

> **Important:** PWA features (service worker, installability, offline) only fully work  
> in a production build or `npm run preview`. Use `preview` to test before deploying.

## Installing on Your Phone

### Android (Chrome)
1. Open the app URL in Chrome
2. Tap the **"Install"** banner that appears, or tap the three-dot menu → "Add to Home Screen"
3. The app appears on your home screen and works fully offline

### iOS (Safari)
1. Open the app URL in Safari (must be Safari — Chrome on iOS cannot install PWAs)
2. Tap the **Share** button (box with arrow)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add** — the app icon appears on your home screen

### Desktop (Chrome / Edge)
1. Open the app URL
2. Click the **install icon** in the address bar (looks like a computer with a download arrow)
3. Click **Install** — the app opens in its own window, no browser chrome

## Deploying (so it lives on the web)

### Option A — Vercel (recommended, free)
```bash
npm install -g vercel
npm run build
vercel --prod
```

### Option B — Netlify (free)
```bash
npm run build
# Drag the `dist/` folder to app.netlify.com/drop
```

### Option C — GitHub Pages
```bash
# In vite.config.js, set: base: '/your-repo-name/'
npm run build
# Push `dist/` to the `gh-pages` branch
```

> After deploying, visit the URL on your phone's browser to install as a home screen app.

## Data & Privacy

All your application data is stored **locally in your browser** using IndexedDB.  
Nothing is sent to any server. Exporting CSV is the only way data leaves your device.
